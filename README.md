# 🌱 The Germinator 🦾

The Germinator is a smart plant germination station designed to optimize the growth of seedlings. It 
combines photobiology, temperature control, and temperature, humidity, and soil moisture monitoring.

## Hardware

- [Raspberry Pi Pico W 2](https://vilros.com/products/raspberry-pi-pico-2w)
- [AHT20 Temperature and Humidity Sensor](https://www.adafruit.com/product/4566)
- [I2C Soil Moisture Sensor](https://www.adafruit.com/product/4026)
- [WS2812 8x32 Module Matrix Screen 5V](https://www.aliexpress.us/item/3256803715519232.html)
- [iPower Seedling Heat Mat](https://www.growsupplystore.org/ipower-seedling-heat-mat-heating-pad-48x20-with-digital-thermostat/)
